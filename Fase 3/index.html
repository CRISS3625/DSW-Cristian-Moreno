El proposito de esta web aclarar y enseñar cada una de las etiquetas basicas de HTML, con su respectivos atributos y valores. Mediante esta web veremos un claro ejemplo de lo que se puede hacer con estas etiquetas.

Las Etiquetas Basicas de HTML: Estas etiquetas de HTML, son fragmentos de codigo que permiten crear elementos HTML, basicamente son estructuras básicas del lenguaje de programación HTML en el que se escriben las páginas web porque es el que interpreta los navegadores. Atributos y Valores: Los atributos constan de un nombre y un valor separados por un signo igual (=), con el valor entre comillas dobles. Aquí hay un ejemplo style="color:black" ,. Los atributos se agregan a una etiqueta para proporcionar al navegador mas informacion sobre como debe comportarse una etiqueta.
